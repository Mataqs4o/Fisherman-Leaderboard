# Fisherman Leaderboard

A simple web application for tracking and ranking fishermen based on their catches. Users can submit scores and view the current leaderboard.

## About

The Fisherman Leaderboard allows users to add their results and compare them with others. The data updates dynamically so the rankings stay current.

## Features

- View a ranked list of fishermen  
- Submit new scores  
- Refresh and update leaderboard data  


### Usage

Open the index.html file in your browser, or run the project using a local development server.
